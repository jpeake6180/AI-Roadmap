How to Use This Roadmap

Work through the phases in order. Pair each technical topic with the matching math, and ship a small project every 1–2 weeks.

  • Phase 0: Refresh Python, NumPy, Pandas & basic stats.
  • Phase 1: Core ML with Andrew Ng + Kaggle.
  • Phase 2: Deep learning with fast.ai.
  • Phase 3: AI engineering & LLMs.

Everything is designed to feed into a portfolio that makes you hireable as an AI/ML engineer.
